Pricey Walkie-talkie



Matin and Kianoosh have bought a pair of walkie-talkies. Yet there are some limitations. They can only send n words through them to the other. And some of the words mean the same thing.
Kianoosh the ith word has a cost of ai for sending. And he has a message he wants to send. He can replace each word with any other word with the same meaning. Help kianoosh find the minimum cost to send his message to Matin.
You must fill in the solve function which takes in n, m, a list of words, a list of costs, and a list of groups and Kianoosh's message.

n is the total number of words.

m is the amount of words in Kianoosh's message.

The list of all the n words and a list of respective prices.

A list of groups which is a list of tuples where each tuple is the indices (Index in the above list of words) of words with the same meaning.

And finally a list of the words in Kianoosh's message.

The function must output a single integer. The minimum cost.

It is known that 1 ≤ n ≤ 10000, 1 ≤ m ≤ 10000

Examples

Example 1

n = 2
points = [(2, 1), (1, 2)]
output = 1

Example 2

n = 2
points = [(2, 1), (4, 1)]
output = 0
