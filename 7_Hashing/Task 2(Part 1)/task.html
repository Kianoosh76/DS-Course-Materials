Task:
The smallest period of a sequence is defined as the smallest positive (nonzero) integer T such that for every i < len(seq) - T we have
seq[i] = seq[i + T]
Complete smallest_period function such that for every string str, it returns the str's smallest period.

Input: seq (A string of lower case characters)
Output: str's smallest period

Limitations:
seq size is not greater than 10^6
You can solve this problem in linear time using hash
