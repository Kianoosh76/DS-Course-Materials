پیش‌نیاز:
آشنایی با داده‌ساختار لیست پیوندی

دو node متمایز در یک لیست پیوندی دوسویه را با هم جابه‌جا کنید. توجه داشته باشید که این دو node ممکن است پشت سر هم باشند.
انتظار می‌رود پیچیدگی تابع شما از (O(1 باشد.
nodeها به شکل زیر تعریف شده‌اند: 

  class Node:
    def __init__(self, data, next_node=None, prev_node=None):
      self.data = data
      self.next = next_node
      self.prev = prev_node
self.prev برای node ابتدایی و self.next برای node انتهایی برابر None هستند.
توجه داشته باشید که برای قبول شدن تابعتان، نباید در آن node جدیدی تعریف کنید یا مقادیر nodeها را تغییر دهید.
